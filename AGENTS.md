# AGENTS.md

## Project

Marketing website for WLM Plomberie built with Next.js, TypeScript, Tailwind CSS, App Router.

## Goals

- Fast local-business showcase website
- Clean SEO structure
- Mobile-first
- Simple, maintainable code
- No overengineering

## Tech rules

- Use Next.js App Router
- Use TypeScript everywhere
- Use functional React components
- Prefer Server Components by default
- Use Client Components only when needed
- Use Tailwind CSS for styling
- Keep components small and reusable
- Avoid unnecessary libraries

## Project structure

- `src/app` for routes
- `src/components/layout` for header/footer
- `src/components/sections` for homepage sections
- `src/components/ui` for small reusable UI pieces

## Coding preferences

- Prioritize readability over abstraction
- Do not introduce complex state management
- Do not add animations unless requested
- Keep forms simple
- Favor semantic HTML and accessibility
- Optimize for Core Web Vitals and local SEO

## Business context

This is a plumbing and air conditioning showcase website for a local business in Nice area.
Main priorities:

- clickable phone number
- strong call-to-action
- trust elements
- service clarity
- Google reviews section
- contact form
